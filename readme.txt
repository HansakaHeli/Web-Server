Execution instructions -: 
	1) From commandline/terminal go this folder. (the folder that contains htdocs folder and java files).
	2) The main method is in the SimpleWebServer.java file. So compile it by running "javac SimpleWebServer.java".
	3) Then, to run the program, execute "java SimpleWebServer". This will start the server on port 2728.
	4) In the browser, type "locahost:2728" to start the server.
	5) It will server the "index.php" file in the htdocs folder.
